## 关于

TasmoAdmin（前称 SonWEB）是一个管理网页界面，用于集中管理您所有的 Sonoff-Tasmota 刷写设备。
它的一些功能包括：

- 扫描您的网络并自动添加您的设备
- 快速轻松查看您所有设备的状态
- 从一个地方配置您所有的设备
- 通过无线方式一次性向一个或多个设备发送固件更新
- 可以为您自动下载最新的固件
