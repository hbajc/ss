看起来像Sonarr，闻起来也像，但这是为音乐而制作的。

## 关于

[Lidarr]是一个音乐收藏管理工具，适用于Usenet和BitTorrent用户。它可以监控多个RSS源，获取您最喜欢艺术家的新曲目，并抓取、排序和重命名它们。它还可以被配置为在更高质量的格式可用时，自动升级已经下载的文件的质量。
