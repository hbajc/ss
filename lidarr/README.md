看起来像 Sonarr，但专为音乐而设计。

## 关于

[Lidarr] 是一个为 Usenet 和 BitTorrent 用户设计的音乐收藏管理器。它可以监控多个 RSS 源，以获取您最喜欢的艺术家的新曲目，并会抓取、排序和重命名它们。它还可以配置为在有更高质量格式可用时，自动升级已下载文件的质量。

[Lidarr]: https://lidarr.audio/
