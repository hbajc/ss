## 关于

[AppDaemon][appdaemon] 是一个松耦合、多线程、沙箱式的 Python 执行环境，用于为 Home Assistant 家庭自动化软件编写自动化应用。它还提供了一个可配置的仪表板（HADashboard），适合用于壁挂式平板电脑。
