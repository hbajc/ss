## 关于

[AppDaemon][appdaemon] 是一个松散耦合、多线程、沙箱化的 Python 执行环境，用于编写 Home Assistant 家庭自动化软件的自动化应用程序。它还提供了一个可配置的仪表板（HADashboard），适合墙面安装的平板电脑。
