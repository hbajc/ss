## 关于

[Radarr] 是一个用于Usenet和BitTorrent用户的电影收藏管理器。它可以监控多个RSS源的新电影，并与客户端和索引器进行交互，以抓取、排序和重命名电影。当有更好的格式可用时，它还可以配置为自动升级库中现有文件的质量。
