## 关于

[Radarr] 是一款用于 Usenet 和 BitTorrent 用户的电影收藏管理器。它可以监控多个 RSS 源以获取新电影，并将与客户端和索引器接口来抓取、排序和重命名电影。当有更高质量的格式可用时，它也可以配置为自动升级库中现有文件的质量。
