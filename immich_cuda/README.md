## 关于

基于Web的文件浏览器。
该插件基于来自 imagegenius 的 [docker 镜像](https://github.com/imagegenius/docker-immich)。

## 配置

PostgreSQL 可以是内部的或外部的。

```yaml
    "PGID": "int",
    "PUID": "int",
    "TZ": "str?",
    "cifsdomain": "str?",
    "cifspassword": "str?",
    "cifsusername": "str?",
    "data_location": "str",
    "localdisks": "str?",
    "networkdisks": "str?",
    "DB_HOSTNAME": "str?",
    "DB_USERNAME": "str?",
    "DB_PORT": "int?",
    "DB_PASSWORD": "str?",
    "DB_DATABASE_NAME": "str?",
    "JWT_SECRET": "str?"
```

## 安装

该插件的安装相当简单，与安装任何其他 Hass.io 插件没有不同。

1. [将我的 Hass.io 插件库添加到您的 Hass.io 实例][repository]。
1. 安装该插件。
1. 单击 `保存` 按钮以存储您的配置。
1. 启动插件。
1. 检查插件的日志以查看一切是否正常。
1. 仔细配置插件以符合您的偏好，具体请参见官方文档。

请注意，您需要安装一个单独的 PostgreSQL 插件才能连接数据库。您可以在我的仓库中提前安装 PostgreSQL 插件。
请注意在启动之前更改密码；它在启动后不会更改。
