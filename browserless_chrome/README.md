## 关于

---

[Browserless chrome](https://github.com/browserless/chrome) 是一个允许远程客户端连接、控制和执行无头工作的网络服务。
该插件基于 Docker 镜像 https://hub.docker.com/r/browserless/chrome/

## 安装

---

安装这个插件非常简单，与安装其他插件没有区别。

1. 将我的插件库添加到您的 home assistant 实例中（在上方右侧的管理员插件商店中，或点击下面的按钮，如果您已配置我的 HA）
   [![打开您的 Home Assistant 实例，并显示添加插件库对话框，填写特定的库 URL。](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Falexbelgium%2Fhassio-addons)
2. 安装此插件。
3. 点击 `保存` 按钮以存储您的配置。
4. 将插件选项设置为您喜欢的方式
5. 启动插件。
6. 检查插件的日志以查看是否一切正常。
7. 打开 WebUI 并调整软件选项
