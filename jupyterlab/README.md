## 关于

JupyterLab 是一个开源网络应用，允许您创建和共享包含实时代码、方程式、可视化和叙述文本的文档。使用场景包括：数据清理和转换、数值模拟、统计建模、数据可视化、机器学习等等。

此插件运行 JupyterLab，这是 Project Jupyter 的下一代用户界面。它是一个可扩展的环境，用于交互式和可重现的计算，基于 Jupyter Notebook 和架构。
