# Home Assistant 插件：邮件过滤器

[![发布][release-shield]][release] ![项目阶段][project-stage-shield] ![项目维护][maintenance-shield]

Rspamd 邮件过滤器，带有可选的 clamav 防病毒功能...
请小心！clamav 使用大量内存！
不建议在内存低于 6GB 的设备上使用。

## 关于

重要提示：此插件需要安装邮件服务器插件！

此插件为实验性，提供邮件服务器的邮件过滤功能。
