# Home Assistant 插件：邮件过滤器

[![Release][release-shield]][release] ![Project Stage][project-stage-shield] ![Project Maintenance][maintenance-shield]

Rspamd 邮件过滤器，带可选的 clamav 杀毒软件...
请小心！clamav 使用巨大的内存！
不建议在内存低于 6GB 的设备上使用。

## 关于

重要：该插件要求安装邮件服务器插件！

该插件是实验性的，为您的邮件服务器提供邮件过滤器。
