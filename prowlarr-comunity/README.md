## 关于

[Prowlarr]是一个基于流行的*arr .net/reactjs基础栈构建的索引器管理器/代理，用于与您的各种PVR应用程序集成。Prowlarr支持对Torrent Tracker和Usenet Indexer的管理。它与Lidarr、Mylar3、Radarr、Readarr和Sonarr无缝集成，提供对您的索引器的完整管理，无需每个应用单独设置索引器（它会自动处理所有）。
