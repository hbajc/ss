## 关于

Matter Python WebSocket 服务器，用于 Home Assistant 核心。Matter（前称为基于 IP 的连接家居或 CHIP）是一种基于 IPv6 的智能家居标准。这个附加组件提供了一个 Matter 控制器，允许您管理和控制 Matter 设备。与此服务器进行通信的 Home Assistant 核心集成通过 WebSocket 进行。
