UniFi网络应用程序允许您使用Web浏览器管理您的UniFi网络。

## 关于

此插件运行Ubiquiti Networks的UniFi网络应用程序软件，它允许您通过Web浏览器管理您的UniFi网络。该插件提供了一键安装和运行解决方案，使Home Assistant用户能够轻松地启动、运行和更新他们的网络。

此插件支持所有Home Assistant支持的架构，包括Raspberry Pi。
