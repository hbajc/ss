一个用于美观分析和监控的开放平台。

## 关于

适用于所有您度量指标的分析平台。

Grafana 允许您查询、可视化、警报和理解您的度量指标，无论它们存储在哪里。创建、探索和分享仪表板。通过迷人而引人入胜的图表和其他数据可视化，了解您的家庭自动化系统。

将此附加组件与 InfluxDB 附加组件结合使用，以获取对您家非常强大的见解。
