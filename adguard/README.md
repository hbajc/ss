## 关于

AdGuard Home 是一个网络范围的广告与跟踪器阻止 DNS 服务器，具有
家长控制（成人内容阻止）功能。它的目的是让
您控制整个网络及所有设备，并且不需要
使用客户端程序。

AdGuard Home 提供一个美观、简单且功能丰富的 web 界面，以
便于管理过滤过程及其设置。
