## 关于

FlexGet 是一款多功能的自动化工具，适用于所有媒体。请查看 https://flexget.com/

默认密码是 'homeassistant123'，或者可以从插件选项中设置

## 需求

## 配置

    "PUID": 自定义用户（默认是 root）
    "PGID": 自定义用户（默认是 root）
    "WebuiPass": webui 密码定义
    "FG_PLUGINS": 插件添加
    "FG_LOG_LEVEL": 日志级别（critical|error|warning|info|verbose|debug|trace）

## 启动

- 启动插件。稍等片刻，检查日志以查看是否有任何错误。
